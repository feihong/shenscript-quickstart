# Feihong's ShenScript quickstart

Getting started with [ShenScript](https://github.com/rkoeninger/ShenScript).

## Prerequisites

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash
    npm install -g yarn

## Install

    yarn add shen-script

## Commands

Run the repl

    yarn repl
